# Claude Tag — AI员工入驻Slack

Claude Tag是Anthropic发布的功能：在Slack频道中 @Claude 就像艾特一个AI同事——它常驻在组织的协作环境里，理解上下文、异步执行任务、主动提醒。^[raw/transcripts/BV1QPjo69Eqv.md]

## 核心特性

1. **上下文驻扎**：你不需要再把文档/上下文粘贴给AI——它跟你看到的是同一个频道的信息
2. **多人共享**：频道里所有人都可以@它，像新来的同事一样
3. **异步执行**：交给它任务后可以在后台完成，你去干别的
4. **持续进化**：了解越多组织上下文，对组织越懂
5. **主动提醒（Ambient）**：主动感知环境，如到周末提醒写周报

## 对组织的影响

Claude Tag是**组织的助手**而非个人助手：
- 需配权限指定它访问哪些频道
- 把沉淀在人脑和散落文档中的组织知识存储下来（类似知识库自动建设）
- 新员工不用从头培训，老员工离职知识不走

## 未来管理者的新角色

> "管理者的工作可能更多成为**上下文的工程设计师**——你给AI配哪些环境、让它了解哪些频道、它可以access哪些工具，这些你要精心设计和准备。"

同时需要权责分明：谁负责判断、执行、审核校验。
